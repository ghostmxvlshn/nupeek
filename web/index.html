nupeek type --package Azure.Messaging.ServiceBus \
  --type Azure.Messaging.ServiceBus.ServiceBusSender \
  --out deps-src

Install

Pick one command and go.

Global tool for regular setup, or curl for quick bootstrap.

.NET global tool

Best for local machines.

bash 
dotnet tool install -g Nupeek
nupeek --help
curl installer

Fast setup for ephemeral terminals.

bash 
curl -fsSL https://raw.githubusercontent.com/ghostmxvlshn/nupeek/main/install.sh | bash

Prefer explicit download?

bash 
curl -fsSL https://raw.githubusercontent.com/ghostmxvlshn/nupeek/main/install.sh -o install.sh
bash install.sh
